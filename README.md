# BigWalkTest
BigWalk 과제 면접

## #1 요구사항
BigWalk 앱의 특정화면을 Clone하기
- (구현완료) (필수) "캠페인 리스트" 기능 구현
- (구현완료) (필수) "전체" 카테고리만 구현
- (구현완료) (선택) 공개형, 그룹형 필터링 (디자인자유)
- (구현완료) (선택) Firebase 통한 SNS 로그인 (디자인자유)
- (구현완료) (선택) "내가 참여한 캠페인" 기능 구현
- (구현완료) (선택) 페이지네이션 기능

## #2 개발환경
Kotlin, MVVM, Retrofit2
